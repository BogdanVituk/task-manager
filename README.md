# Система управління завданнями співробітників

## Опис проєкту
Дана система призначена для постановки, контролю та виконання завдань співробітниками підприємства.  
Метою проєкту є підвищення продуктивності праці, прозорості бізнес-процесів та ефективного контролю термінів виконання завдань.

Система дозволяє керівникам створювати та відслідковувати завдання, а співробітникам — переглядати, виконувати та оновлювати статуси своїх завдань.

## Основний функціонал
- створення та редагування завдань;
- призначення завдань співробітникам;
- контроль статусу виконання завдань;
- встановлення та контроль дедлайнів;
- перегляд списку активних та виконаних завдань;
- прозорий контроль процесу виконання завдань.

## Архітектура
Проєкт реалізований з використанням багатошарової архітектури:

- **DAL (Data Access Layer)** — рівень доступу до даних, відповідає за роботу з базою даних;
- **BLL (Business Logic Layer)** — рівень бізнес-логіки, містить правила обробки даних та логіку роботи системи.

Такий підхід забезпечує:
- розділення відповідальностей;
- зручність підтримки та масштабування;
- можливість повторного використання коду.

## Технології
- **Мова програмування:** C#
- **Платформа:** .NET
- **Архітектура:** DAL / BLL
